# Pomoron

Simple pomodoro technique timer

## Development

```sh
$ git clone https://github.com/saitoxu/Pomoron.git
$ cd Pomoron
$ yarn install
$ electron main.js
$ yarn dev # another shell
```

## Release

```sh
$ electron-packager . Pomoron --platform=win32 --arch=x64 --electron-version=1.6.10
```
